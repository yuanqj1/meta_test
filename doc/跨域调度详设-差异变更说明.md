## 0. 顶层职责重划（一图看懂）

```text
        ┌─────────────────────────────────────────────────────────────┐
        │              旧 MVP-1.5 (ctld 主导路由决策)                  │
        ├─────────────────────────────────────────────────────────────┤
        │ ctld:  识别候选 → hold → 决策(SendTo) → 写 remote_cluster   │
        │           → 写 remote_partition → 限流(MaxJobs)              │
        │           → ACL(AllowApp, comment "allow_remote")            │
        │           → 转发(target_cluster/partition 已确定)            │
        │ broker:接单 → stage → 远端 sbatch → 状态同步                │
        └─────────────────────────────────────────────────────────────┘

        ┌─────────────────────────────────────────────────────────────┐
        │              新 v2 (broker 承担路由能力)                     │
        ├─────────────────────────────────────────────────────────────┤
        │ ctld:  识别候选(只看 AllowRemote+remote_allowed)             │
        │           → hold → 转发 (不传 target)                        │
        │ broker:接单 → route_decide(routes.conf, priority/cap)        │
        │           → for 候选 in 候选列表:                            │
        │               broker→broker test-only(8018) → 成功 break    │
        │           → 全部失败 → 回 ctld NO_VIABLE_ROUTE → 释放       │
        │           → stage → 远端 sbatch (sudo-u 或 root --uid)      │
        │           → UPDATE_REMOTE_STATE 首次回写 cluster/partition   │
        │           → 状态同步                                          │
        └─────────────────────────────────────────────────────────────┘
```

| 子问题 | 旧承担方 | 新承担方 | 真相源 |
|---|---|---|---|
| Q1 选目标集群 | ctld partition.SendTo | broker route_decide | broker routes.conf |
| Q2 选目标分区 | 同上 | 同上 | 同上 |
| Q3 用户映射 | broker.conf LocalUser= | 不变 | broker.conf LocalUser= |
| Q4 软件路径 | broker lookup_software.sh | 不变 | 同上 |
| Q5 应用准入 | ctld partition.AllowApp | broker routes.conf match_app | 同上 |
| Q6 远端准入 | 不做 | broker test-only 探测 | 远端 ctld (slurm_job_will_run) |
| Q7 在途配额 | ctld partition.MaxJobs | broker routes.conf cap | 同上 |
| Q8 用户授权 | user/assoc.comment "allow_remote" 子串 | user/assoc.remote_allowed 字段 | sacctmgr |
